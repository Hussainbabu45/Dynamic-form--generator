# Dynamic Form Generator

A dynamic form generator that creates a styled, functional form in real-time based on a JSON schema. This project features a JSON editor and a live form preview side by side, offering an intuitive interface for developers.

## Features
- **Real-Time Updates:** See changes in the form preview as you edit the JSON.
- **JSON Editor:** Includes syntax highlighting, validation, and error messages for invalid JSON.
- **Responsive Design:** Mobile-friendly interface with a stacked layout on smaller screens.
- **Form Validation:** Supports required fields, custom validation messages, and error states.
- **Submit Functionality:** Logs submitted data to the console and displays a success message.

## Technologies Used
- **React 18+**: For building the user interface.
- **TypeScript**: Ensures type safety and better code maintainability.
- **Tailwind CSS**: For a clean and responsive design.
- **React Hook Form**: Simplifies form validation and management.
